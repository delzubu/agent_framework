# SDK Overlay Documentation

SDK pages are generated by Sphinx from source code. Use this directory for human-written narrative that should be inserted into generated SDK pages.

## Mapping Rules

Module overlay:

```text
docs/sdk-overlays/agent_framework/host.md
```

is inserted into:

```text
docs/sdk-source/api/agent_framework/host.rst
```

Class overlay:

```text
docs/sdk-overlays/agent_framework/host/AgentHost.md
```

is inserted into:

```text
docs/sdk-source/api/agent_framework/host/AgentHost.rst
```

## What Belongs Here

- Purpose and responsibility of a module or class.
- Usage patterns.
- Lifecycle notes.
- How the API fits into the wider runtime.
- Common mistakes and design guidance.

## What Belongs in Source Docstrings

- Public API facts.
- Parameter descriptions.
- Return values.
- Exceptions.
- Short behavior notes.
- Small examples that are tightly coupled to implementation.

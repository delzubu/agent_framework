# Getting Started

Who this is for: developers installing and running `agent_framework` for the first time.

## Goal

Install the package, configure a model provider, and run a minimal agent or headless completion.

## Outline

1. Install the package.
2. Configure provider credentials.
3. Run a simple command.
4. Inspect output and traces.

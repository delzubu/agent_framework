# agent_framework Documentation

`agent_framework` is a markdown-defined agent runtime, orchestration host, tracing toolkit, and evaluator utility set for building and testing agentic systems.

This documentation site is the central hub for learning what the project is, building with it, understanding the design, finding examples, and contributing.

# CLI Reference

Who this is for: users running the runtime or evaluator from the command line.

## Commands

- `python -m agent_framework`
- `python -m agent_framework_evaluator web`
- `python -m agent_framework_evaluator run`
- `python -m agent_framework_evaluator evaluate`

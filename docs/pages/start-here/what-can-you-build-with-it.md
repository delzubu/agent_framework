# What Can You Build With It?

Who this is for: readers looking for practical use cases.

## Use Cases

- Internal assistants for repeatable workflows.
- Tool-using agents for file, API, or MCP-backed tasks.
- Multi-agent workflows with explicit delegation.
- Agent evaluation suites and regression tests.
- Provider-agnostic LLM applications.
- Debuggable prototypes for agentic runtime design.
